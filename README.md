# Simple To-Do CLI

A simple command-line to-do list manager.

## Features
- Add tasks
- Mark tasks as done
- Show all tasks

## How to Run

1. Clone the repository:
    ```bash
    git clone https://github.com/0xHawre/simple-todo-cli.git
    cd simple-todo-cli
    ```

2. Run the application:
    ```bash
    python3 todo.py
    ```

3. Follow the prompts to manage your tasks.
